# FitPlay

Turn your workout into an epic gaming adventure powered by AI.

## About

FitPlay is an AI-powered fitness gaming platform that transforms exercise into immersive games. Your body becomes the controller - no equipment needed, just a camera and space to move.

## Features

- **AI Pose Detection**: Real-time tracking of body joint points
- **Gaming Experience**: Boss battles, challenges, and daily quests
- **Progress Tracking**: XP, levels, streaks, and detailed analytics
- **Multiplayer**: Compete with friends in real-time workout battles
- **No Equipment Needed**: Works with just your device's camera

## Tech Stack

- **Frontend**: React + TypeScript
- **Styling**: Tailwind CSS + Shadcn UI
- **Build Tool**: Vite
- **Routing**: React Router
- **State Management**: React Hooks
- **Theme**: Next Themes (Dark/Light mode)

## Getting Started

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Run development server**
   ```bash
   npm run dev
   ```

3. **Open in browser**
   - Navigate to `http://localhost:8081`
